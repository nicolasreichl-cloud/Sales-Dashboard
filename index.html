    🏗
    MVP Architecture & Roadmap. This section documents the data model, workflows, and expansion plan for the Outreach Automation Dashboard.

    📐 Data Model

    Primary Sources:

    • HubSpot Deals — Pipeline stage "Nurture" (149 deals)

    • HubSpot Companies — Top 120 utilities (119 companies)

    • HubSpot Owners — Sales team assignment

    • Notion Purple Sheets — Deep customer context per utility

    Key Fields Used:

    • Deal: name, stage, owner, amount, last contacted, close date

    • Company: name, country, top_120, contacted count, last activity, domain

    • Purple Sheet: status, country, contacts, priority, solution proposed

    ⚙️ Workflow Logic

    Module 1 — Nurture Follow-up:

    • Priority = f(days_silent, deal_value, has_purple_sheet)

    • Critical: >180 days silent OR never contacted + high value

    • High: 90–180 days silent

    • Medium: 30–90 days · Low: <30 days

    Module 2 — Triggers:

    • Stale re-engagement: High-value deals dormant >120 days

    • Close date passed: Deals past expected close

    • Multi-deal: Companies with multiple nurture deals

    Module 3 — Coverage:

    • Whitespace = top_120 + num_contacted = 0
